# Alt_NC_Bathroom
An Arduino-Powered alternative bathroom door handle for North Carolina

You need:
* Arduino Board
* LCD display
* Breadboard
* Couple resistors
* Lots of wires
* A door handle

Main code: [restroom.ino](restroom.ino)
